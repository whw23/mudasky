# Header UI 重新设计

## 概述

将现有的红色顶栏 + 白底导航栏重新设计为白底毛玻璃浅色系风格，提升现代感和视觉美感。

## 设计决策

### 风格：白底毛玻璃

- **顶部信息栏**：深色半透明底（`rgba(26,26,46,0.9)` + `backdrop-filter: blur(12px)`），包含热线电话、语言切换、登录按钮
- **主导航栏**：白色半透明底（`rgba(255,255,255,0.85)` + `backdrop-filter: blur(20px)`），滚动时固定（sticky）并增强毛玻璃效果
- **品牌名**：深色字体，不再用红色
- **活跃导航项**：红色文字 + 底部指示线
- **导航菜单项**：`whitespace-nowrap` 禁止换行

### 色彩调整

- 保持现有 primary 红色（`oklch(0.45 0.2 25)`）作为强调色
- 顶部栏从纯红改为深色系
- 整体氛围从"红色教育机构"转向"现代专业服务"

### 组件

- Header.tsx 重写：两层布局（顶部信息栏 + 毛玻璃导航栏）
- 导航栏 sticky 定位，滚动时顶部信息栏收起
- 移动端：汉堡菜单保持，菜单面板加毛玻璃效果

### 不涉及

- Banner 组件（保持不变）
- 页面内容区域
- 管理后台/用户中心布局
- Footer
