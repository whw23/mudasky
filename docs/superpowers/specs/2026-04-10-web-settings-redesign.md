# 网页设置重构 + Header 改版设计

## 概述

将后台管理中分散的页面内容管理（文章/案例/分类/院校/设置）整合为统一的"网页设置"入口，以网站预览形式呈现，支持所见即所得编辑。同时改版 Header 样式、合并联系我们页面。

## 一、Header 改版

### 桌面端（≥768px）

两行结构：

- **顶栏**：左侧 Logo 红色方块 + 大字号品牌名(22px, font-weight 800) + 标语；右侧服务热线(红色加粗) + 用户名 + 语言切换
- **导航栏**：只放导航链接，无品牌名。当前页红色文字 + 红色下划线，其他灰色

### 移动端（<768px）

一行结构：

- Logo 方块 + 品牌名 + 标语（缩小）| 右侧电话图标（可点击拨号）+ 汉堡菜单
- 展开菜单：卡片式列表，底部附服务热线栏（红色背景）

### 断点变更

- 桌面/移动切换断点从 `lg:1024px` 改为 `md:768px`
- 9 个导航项在 768px 宽度下足够横向排列

### 导航栏顺序

```text
网站首页 | 院校选择 | 出国留学 | 申请条件 | 成功案例 | 签证办理 | 留学生活 | 新闻政策 | 关于我们
```

## 二、页面调整

### 联系我们页面取消

- 删除 `app/[locale]/(public)/contact/` 路由
- 联系方式（地址/电话/邮箱/微信/办公时间）合并到"关于我们"页面底部作为一个区块
- ContactForm 表单取消，用户留联系方式改为引导注册
- 导航栏从 10 项减为 9 项

### 地图模块

- 不在联系方式中使用
- 预留给院校详情页（展示学校地理位置）

## 三、Footer

- 保持当前浅色方案不变（`bg-muted/50`，4 列布局）
- Footer 中"联系我们"快捷链接移除或改为"关于我们"

## 四、后台管理重构 — 网页设置

### 侧边栏变更

原有菜单项：

```
仪表盘 | 用户管理 | 角色管理 | 文章管理 | 案例管理 | 分类管理 | 院校管理 | 设置
```

新菜单项：

```
仪表盘 | 用户管理 | 角色管理 | 网页设置
```

- 文章管理、案例管理、分类管理、院校管理、设置 → 全部整合进"网页设置"
- 权限标识：`admin.settings.*`（沿用）

### 网页设置页面结构

整个页面 = 完整网站预览：

```
┌─────────────────────────────────────┐
│ Header（可编辑）                      │
├─────────────────────────────────────┤
│ 导航栏（点击切换页面预览）              │
│ 网站首页 | 关于我们 | 出国留学 | ...   │
├─────────────────────────────────────┤
│                                     │
│ 当前页面内容预览（可滚动）              │
│ 每个区块 hover 显示编辑按钮            │
│                                     │
├─────────────────────────────────────┤
│ Footer（可编辑）                      │
└─────────────────────────────────────┘
```

### 交互规则

1. **导航栏**：复用公共网站导航样式，点击切换显示对应页面的预览内容
2. **可编辑区块**：hover 时显示蓝色虚线边框 + 编辑按钮（右上角），不 hover 时完全隐藏
3. **编辑弹窗**：点击编辑按钮弹出 ConfigEditDialog，保存后预览实时刷新
4. **数据驱动区块**：文章列表、案例、院校等区块，点击编辑按钮打开对应的管理表格（内嵌或弹窗）
5. **页面链接禁用**：预览中的所有链接和按钮不可点击，不会触发跳转
6. **Header/Footer 共享**：在任意页面预览中编辑 Header/Footer，全站生效

### 各页面预览内容

| 导航项 | 预览包含的可编辑区块 |
|--------|---------------------|
| 网站首页 | Hero 横幅、统计数据、服务介绍、留学国家、最新资讯(自动拉取)、CTA |
| 院校选择 | 介绍、院校列表(管理表格)、选校建议 |
| 出国留学 | 概述、德语项目、项目对比、相关文章(自动拉取) |
| 申请条件 | 各国要求、语言要求、材料清单、时间线、相关文章(自动拉取) |
| 成功案例 | 统计、案例列表(管理表格) |
| 签证办理 | 流程、材料清单、时间线、注意事项、相关文章(自动拉取) |
| 留学生活 | 介绍、生活指南、城市指南、相关文章(自动拉取) |
| 新闻政策 | 分类管理、文章列表(管理表格) |
| 关于我们 | 公司故事、使命愿景、合作机构、团队介绍、统计数据、联系方式 |

### 复用现有组件

- `EditableOverlay` — 已有的 hover 编辑浮层组件，直接复用
- `ConfigEditDialog` — 已有的配置编辑弹窗，直接复用
- `ArticleTable` / `CaseTable` / `UniversityTable` / `CategoryTable` — 现有管理表格组件，内嵌到对应页面预览中
- 各公共页面的 Section 组件 — 复用作为预览渲染

### 通用配置

不属于任何页面的配置（如国家代码编辑器）保留为导航栏外的独立入口，或作为页面底部的附加区域。

## 五、不在本次范围内

- 页面正文内容（i18n 翻译文件中的文案）暂不改为后台可编辑
- 导航菜单项暂不支持后台自定义（保持硬编码）
- 子菜单/下拉导航
- 地图组件实现（仅预留）
