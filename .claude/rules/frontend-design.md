## 前端设计规范（旧系统参考）

旧官网截图在 `legacy_pages/` 目录下。

### 统一组件

- **顶部横幅**：红色背景，显示服务热线
- **导航栏**：Logo + 栏目（网站首页、关于我们、出国留学、院校选择、申请条件、成功案例、签证办理、留学生活、新闻政策、联系我们）
- **Banner**：每个页面都有大图 Banner + 页面标题（中英文）
- **底部**：深灰背景，公司介绍 + 联系方式（地址、电话、邮箱）+ 微信二维码 + ICP 备案

### 三种页面模板

1. **首页**：多板块拼接（关于我们 + 精选服务 + 院校选择 + 师资 + 热门申请 + 新闻中心 + 教育文化）
2. **文章列表页**：左侧文章列表（缩略图+标题+摘要+日期+分页）+ 右侧推荐/最新文章侧边栏
3. **内容展示页**：内容区 + 子分类 Tab 切换

### 设计风格

- **主色调**：红色（约 #C41A1A）+ 白色 + 深灰底部
- **风格**：正式、稳重的教育机构风格

### 交互规范

- 所有可点击的**文字链接** hover 时变红色
- **文章卡片** hover 时整个卡片背景变红色，文字变白色（红底白字）
- 非 hover 状态：白底黑字
- 导航菜单当前激活项为红色，其他项 hover 时也变红色
