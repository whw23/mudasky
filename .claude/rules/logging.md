## Logging

- 统一使用 JSON 结构化日志
- Python 使用标准库 `logging`
- JS/TS 使用 `console` 或轻量库（如 pino）

## Log Level

| 级别    | 使用场景                                 |
|---------|------------------------------------------|
| DEBUG   | 开发调试信息，生产环境关闭               |
| INFO    | 正常业务流程（用户登录、请求处理等）     |
| WARNING | 异常但不影响运行（重试、降级等）         |
| ERROR   | 需要关注的错误（请求失败、数据库异常等） |
